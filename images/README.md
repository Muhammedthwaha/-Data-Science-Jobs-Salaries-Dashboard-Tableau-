# Dashboard Screenshots and Images

This directory contains screenshots and visualizations from the Data Science Jobs Salaries Dashboard.

## Contents

This folder is intended for:
- Dashboard screenshots
- Individual chart/visualization images
- Infographics and summary visuals
- Presentation materials

## Guidelines

When adding images:
- Use descriptive filenames (e.g., `salary_by_experience_chart.png`)
- Prefer PNG format for screenshots
- Include a brief description if the image needs context
- Keep images at reasonable resolution (optimized for web viewing)

## Usage

Images from this directory can be:
- Embedded in the README.md
- Used in presentations
- Shared in documentation
- Referenced in reports

## Note

Screenshots will be added as dashboard visualizations are created in Tableau.
